# Cognitive Kernel-Pro Paper Website

This repository contains the website for the Cognitive Kernel-Pro research paper.

## Repository

- **GitHub Repository**: https://github.com/bowen31337/cognitive-kernel-pro-paper
- **GitHub Pages URL**: https://bowen31337.github.io/cognitive-kernel-pro-paper/

## Deployment

The website is automatically deployed to GitHub Pages using GitHub Actions. The workflow:

1. Triggers on pushes to the `main` branch
2. Copies `paper.html` to `index.html`
3. Deploys to GitHub Pages

## Local Development

Simply open `paper.html` in your browser to view the website locally.
